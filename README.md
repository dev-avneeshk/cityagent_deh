# 🏙️ City Agent

## 🚀 Overview
City Agent is a real-time urban monitoring system that processes data from multiple sources to detect and analyze city incidents. It filters relevant data, identifies anomalies using statistical techniques, and generates structured incident reports with severity levels.

---

## ⚙️ Features
- Data filtering based on location and keywords
- Fast threshold-based anomaly detection
- Z-score analysis using historical data
- Weighted scoring system for severity classification
- Lightweight SQLite database for storage

---

## 🧠 Workflow
1. Collect data (tweets, sensors, etc.)
2. Filter relevant city data
3. Detect anomalies (threshold + Z-score)
4. Score and label incidents
5. Output structured incident reports

---

## 🏗️ Tech Stack
- Python
- SQLite
- NumPy

---

## 🎯 Goal
To build a smart, scalable system for real-time city monitoring and incident detection.
